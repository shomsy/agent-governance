# PHP Profile

This profile defines reusable PHP engineering guidance for repositories using Agent Harness.

## Required Detail Rules

Load these profile extensions when PHP is selected:

- `.agents/.rules/governance/profiles/languages/php.d/modern-php-attributes-di.md`
- `.agents/.rules/governance/profiles/languages/php.d/code-style.md`
- `.agents/.rules/governance/profiles/languages/php.d/coding-standards.md`
- `.agents/.rules/governance/profiles/languages/php.d/clean-code.md`

## Operating Rule

Prefer modern PHP, strict typing, clear dependency injection, explicit boundaries, behavior-focused tests, secure defaults, and simple call sites.
