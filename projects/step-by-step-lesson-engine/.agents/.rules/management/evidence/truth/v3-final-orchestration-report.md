## 1. Society of Mind (SoM) Lifecycle
Multi-agent orchestration follows the SoM pattern, encapsulating sub-team complexity and optimizing token usage through context compression.

## 2. Delegation & Routing
Routing is profile-driven. Delegation is managed through rigid artifact contracts, ensuring worker agents receive only the necessary context for their specific scope.

## 3. Failure Propagation
The system defines a clear failure propagation model, preventing agents from hallucinating fixes during hard failures and ensuring errors bubble up to the supervisor in a structured format.
