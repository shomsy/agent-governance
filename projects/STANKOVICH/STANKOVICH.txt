# Architecture Manifesto: Minimalist Signal-Driven Web Components Engine

Dobrodošli u tehnički dokument arhitekture našeg `zero-dependency` engine-a. Cilj ovog dokumenta je da pojasni inženjerske i filozofske odluke koje stoje iza sistema, omogućavajući svakom programeru ili arhitekti da tačno razume kako **reaktivnost** i **DOM manipulacija** funkcionišu ispod haube.

---

## 1. Filozofija (Zašto "Architecture 🅱"?)

Većina modernih framework-a (React, Vue) se oslanja na koncept VDOM-a (Virtual DOM) gde se reaktivno stanje prevodi u celokupno novo DOM stablo, a zatim se vrši takozvani *diffing* i *patching* promena.

Ovaj framework obacuje taj teški pristup u korist **Čiste Signal Arhitekture**.

Princip se sastoji iz sledećeg:

1. **Nema Renderovanih Ciklusa:** Komponenta se *crta* u DOM-u **samo jednom** (kroz `mount()`). Ne postoji VDOM pomirenje (reconciliation).
2. **Reaktivnost Kao Prva Klasa Građanin:** Svi podaci žive u *Signalima*, izgrađenim na modelu "pub-sub" (Observer obrazac).
3. **Direktni Binders:** Ovi signali se privezuju direktno na HTML Node elemente posredstvom mikroskopskih efekata (effects). Kada signal promeni vrednost, obnavlja se samo onaj najmanji mogući atribut i element umesto rerenderisanja komponenti.

---

## 2. Prikaz Strukture Fajlova

Za stabilno upravljanje i lako snalaženje, arhitektura je podeljena u funkcionalne blokove nezavisne namene:

```text
/core
  ├── signals.js     # Čist reaktivni observer. Nije svestan postojanja DOM-a.
  ├── dom.js         # Manipulacija nad nativnim elementima (O(1) DOM bind-ovanje).
  ├── component.js   # Encapsulacija Shadow DOM-a i Web Komponente. Upravlja životnim vekom.
  └── builder.js     # Razvojno-prijemčiv, Fluent DSL API lančani konstrukt.
index.js             # Centralni interfejs ka razvoju same aplikacije (Facade layer).
```

---

## 3. Tehnički Stubovi Sistema

### I. Signal Engine (`core/signals.js`)

*SolidJS stil reaktivnosti.* Ne postoji globalno prop-drilovanje.
Sve reaguje oko tri tačke:

- **`signal(value)`**: Wrappovana reaktivna vrednost.
- **`effect(fn)`**: Radnik. Svaki put kada se signal na koji ukazuje ova funkcija (njen dependency graph) promeni, ova funkcija će se reizvršiti.
- **`computed(fn)`**: Derivirani signal koji emituje nove rezultate tek po promeni pratećih signala.

> **Leak-Safe Pravilo (uz ownership disciplinu)**: Sve efekte koji se pokrenu za vreme *mount*-a jedne komponente, centralna matica registruje unutar komponente. Kada browser ukloni komponentu (`disconnectedCallback()`), registrovani efekti se momentalno uništavaju (`stop(runner)`).

### II. Nativni DOM Alati (`core/dom.js`)

Nismo pravili prevođenje (kompajler), već smo ostali pri brutalno čistom API-ju.
Funkcija `h()` kreira direktan **nativan DOM element** `document.createElement()`, dok se ostali `bindXxx` alati oslanjaju na efekte nad tim DOM-om.

```javascript
const clicks = signal(0);
const spanEl = h("span", { class: "value" });

// DOM element je odmah vezan na Signal
bindText(spanEl, clicks);
```

Ova akcija generiše skriveni Effect u pozadini i svaka izmena unutar `clicks.value++` direktno ažurira tekstualni node. Ne postoji nikakvo brisanje ili preklapanje celog roditeljskog drveta!

### III. Web Komponenta i Sigurnost (`core/component.js`)

Klasa `BaseComponent` nadograđuje standardni standard browsera obezbeđujući nativnu izolaciju CSS modula - `ShadowRoot`.

- Pored stila i inkapsulacije, pretvara primitivna polja za unos HTML atributa (npr. `<m-counter step="5">`) direktno u interne signale unutar samog frameworka. Lookupi su asimptotske složenosti O(1) zahvaljujući asocijativnim hash poljima instanciranim pre renderovanja.
- Takođe, implementiran je domorodački nativni mehanizam Dependency Injection-a tj. **Context API** (preko standardnih *Event Bubble* tehnologija) gde unutrašnje strukture lako dohvaćaju spoljašnje Provider elemente ne dirajući signale.

### IV. Fluent DSL Builder (`core/builder.js`)

Kreiranje Web Komponenti kroz čist JavaScript može izgledati nezgrapno zbog količine protokola koju je neophodno ispratiti.
DSL lanac je tu da taj teret oduzme sa inzenjera i dopusti brz prelaz sa prototipa na "Production" nivo kvaliteta.

```javascript
Component.define("my-button")
   .props({
       theme: { type: "string", default: "light", reflect: true }
   })
   .setup(function() {
      // Inicijalizacije logike pre kreiranja DOM-a
   })
   .mount(function(root, ctx) {
      // Izgradnja interfejsa 1 na 1 i umetanje u root (Shadow/HTML) element
   })
   .register();
```

---

## 4. Zaključak

Primenjujući ovu paradigmu, ovaj framwork uspeva da pretekne standardne reaktivne VDOM mehanizme u performansama. Kriterijumi koji važe su:

- *Očuvani mentalni fokus* developera. Nema "čarolija" ni globalnog praćenja.
- Apsolutna O(1) asimptotika u renderu na UI nivo.
- Jednostavnost implementacije bez paketa veličine 14MB. Svega `<450` probranih linija inženjeringa!

# 🅂 Signal-Driven Framework: STANKOVICH

This document defines the core architecture and development patterns for the **Minimalist Signal-Driven Web Components Engine**.

## 1. Core Philosophy

* **Signals Only**: Reactivity is explicit and granular. No Virtual DOM, no Proxies, no global diffing.
* **One-Time Mount**: Components are mounted exactly once. Prop/State updates trigger fine-grained effects that manipulate the DOM directly.
* **Total Ownership**: Every side effect is tied to a component's lifecycle. Memory leaks are avoided by design when following ownership rules.

## Source Of Truth

* Canonical implementation lives in `core/*.js`, `index.js`, `index.d.ts`, `tests/*`, and `examples/*`.
* `component-builder.txt` is a generated merge artifact for packaging/review, not an editable source file.

## 2. Reactive Primitives

* **`signal(val)`**: Returns a reactive container. Access via `.value`.
* **`effect(fn)`**: Executes side effects. Automatically tracks signal dependencies.
* **`computed(fn)`**: Returns a derived signal that only updates when its dependencies change.
* **`stop(runner)`**: Permanently deactivates an effect (and also supports computed signals).
* **`stopComputed(sig)`**: Explicitly detaches a computed from dependencies.

## 3. Component Lifecycle & Ownership

### The `BaseComponent` Rules

1. **Mount Isolation**: DOM is rendered into a `#mountPoint` (inside Shadow DOM) with `display: contents`. Styles are isolated from teardowns.
2. **Auto-Tracking**: Any `effect()` created during the `mount` or `setup` phase is automatically registered for cleanup.
3. **Manual Ownership**: Use `this.own(runner)` to register external effects or binders created outside the initial mount.
4. **Automatic Teardown**: `disconnectedCallback` stops all registered effects and clears the mount point.

## 4. Development Patterns (DX)

### Defining a Component

```javascript
Component.define("my-counter")
  .props({ step: { type: "number", default: 1 } })
  .setup(function() {
    const count = signal(0);
    return { count };
  })
  .mount(function(root, ctx) {
    const btn = h("button", { on: { click: () => ctx.count.value += this.step } });
    bindText(btn, ctx.count);
    root.appendChild(btn);
  })
  .register();
```

### Binders

Binders (`bindText`, `bindAttr`, `bindProp`, `bindClass`) return an effect runner. In `mount()`, they are auto-owned. Outside `mount()`, use `this.own()`:

```javascript
this.own(bindText(el, signal));
```

For composable text children, use `text(sig)`:

```javascript
const line = h("p", {}, "Value: ", text(countSig));
```

### Property vs Attribute

* Use `h("input", { value: "..." })` for initial state.
* Use `bindProp(el, "value", signal)` for live reactive state on inputs.

## 5. Context API

* `provide(ctx, valOrSignal)`: Shares state down the tree. Non-signals are automatically wrapped.
* `consume(ctx)`: Returns a **Signal**. Always reactive.

=== core/builder.js ===
/**
 * core/builder.js
 * 
 * This file gives developers a nice, easy way to describe their components.
 * It's like a menu where you pick what your component has.
 */

import { BaseComponent } from "./component.js";

/**
 * ComponentBuilder lets us chain functions together.
 */
class ComponentBuilder {
    #tag;                // The name of our component (must have a dash like "my-app").
    #props = {};         // The data it can receive.
    #styles = "";        // How it looks.
    #setup = function () { return {}; }; // How it starts up.
    #mount = function () { };           // How it builds its UI.
    #methods = {};       // Things it can DO.
    #onConnected = null; // When it joins the screen.
    #onDisconnected = null; // When it leaves the screen.

    constructor(tag) {
        if (!tag.includes("-")) throw new Error(`Custom element name must include a dash. Got: "${tag}"`);
        this.#tag = tag;
    }

    /** 1. Define what data it takes. */
    props(defs) {
        this.#props = { ...this.#props, ...defs };
        return this;
    }

    /** 2. Define how it looks. */
    styles(styleText) {
        this.#styles = String(styleText ?? "");
        return this;
    }

    /** 3. Define the data and logic (runs once). */
    setup(fn) {
        this.#setup = fn;
        return this;
    }

    /** 4. Define the UI (runs once). */
    mount(fn) {
        this.#mount = fn;
        return this;
    }

    /** 5. Define actions it can perform. */
    methods(methodMap) {
        this.#methods = { ...this.#methods, ...methodMap };
        return this;
    }

    /** 6. Define what happens when it's born. */
    onConnected(fn) {
        this.#onConnected = fn;
        return this;
    }

    /** 7. Define what happens when it dies. */
    onDisconnected(fn) {
        this.#onDisconnected = fn;
        return this;
    }

    /**
     * This final function tells the browser about our new component!
     */
    register() {
        const tag = this.#tag;
        const props = this.#props;
        const styles = this.#styles;
        const setupFn = this.#setup;
        const mountFn = this.#mount;
        const methods = this.#methods;
        const onConnected = this.#onConnected;
        const onDisconnected = this.#onDisconnected;

        // We "bake" all our picked options into a real Class!
        class BuiltComponent extends BaseComponent {
            static props = props;
            static styles = styles;
            static setupFn = setupFn;
            static mountFn = mountFn;
        }

        if (onConnected) BuiltComponent.prototype.onConnected = onConnected;
        if (onDisconnected) BuiltComponent.prototype.onDisconnected = onDisconnected;

        // This makes 'this.someData' act like a shortcut for our Signal's value.
        for (const prop of Object.keys(props)) {
            Object.defineProperty(BuiltComponent.prototype, prop, {
                get() { return this.props[prop].value; },
                set(val) { this.props[prop].value = val; }
            });
        }

        // Put our actions on the component!
        for (const [name, fn] of Object.entries(methods)) {
            if (name in BuiltComponent.prototype) {
                throw new Error(`Method conflicts with existing BaseComponent member: ${name}`);
            }
            BuiltComponent.prototype[name] = fn;
        }

        // Tell the browser!
        if (!customElements.get(tag)) {
            customElements.define(tag, BuiltComponent);
        }

        return BuiltComponent;
    }
}

function coerceFromAttr(type, raw) {
    if (type === "boolean") return raw !== null;
    if (raw === null) return undefined;
    if (type === "number") {
        const n = Number(raw);
        return Number.isFinite(n) ? n : undefined;
    }
    if (type === "json") {
        try { return JSON.parse(raw); } catch { return undefined; }
    }
    return String(raw);
}

function coerceToAttr(type, val) {
    if (type === "boolean") return val ? "" : null;
    if (val === null || val === undefined) return null;
    if (type === "json") return JSON.stringify(val);
    return String(val);
}

function propToAttr(propName) {
    return propName.replace(/[A-Z]/g, (m) => `-${m.toLowerCase()}`);
}

const liveComponents = new Map();

function registerLiveComponent(instance) {
    const tag = instance.localName || instance.constructor?.name || "component";
    let set = liveComponents.get(tag);
    if (!set) {
        set = new Set();
        liveComponents.set(tag, set);
    }
    set.add(instance);
}

function unregisterLiveComponent(instance) {
    const tag = instance.localName || instance.constructor?.name || "component";
    const set = liveComponents.get(tag);
    if (!set) return;
    set.delete(instance);
    if (set.size === 0) liveComponents.delete(tag);
}

export function getComponentStats() {
    const byComponent = {};
    let totalComponents = 0;
    let totalEffects = 0;
    let totalListeners = 0;

    for (const [tag, set] of liveComponents.entries()) {
        let effects = 0;
        let listeners = 0;
        for (const instance of set) {
            const stats = instance.getOwnershipStats();
            effects += stats.effects;
            listeners += stats.listeners;
        }

        byComponent[tag] = { instances: set.size, effects, listeners };
        totalComponents += set.size;
        totalEffects += effects;
        totalListeners += listeners;
    }

    return { totalComponents, totalEffects, totalListeners, byComponent };
}

    scope(fn) {

            return fn.call(this, this);
        } finally {
            setEffectTracker(prev);
        }
    }

    provide(context, value) {
        this._providers = this._providers || new Map();
        const sig = isSignal(value) ? value : signal(value);
        this._providers.set(context.name, sig);

                if (this._providers.has(e.detail.name)) {
                    e.stopPropagation();
                    e.detail.callback(this._providers.get(e.detail.name));
                }
            });

        }
    }

    consume(context) {
        let sig = signal(context.initialValue);
        this.dispatchEvent(new CustomEvent("context-request", {
            detail: { name: context.name, callback: (v) => sig = v },
            bubbles: true, composed: true
        }));
        return sig;
    }

        this.#effects.add(runner);
        return runner;
    }

            listeners: this.#listeners.size

            });

        }

    }

    attributeChangedCallback(attrName, oldValue, newValue) {
        if (oldValue === newValue) return;
        const Ctor = this.constructor;
        const prop = Ctor._attrToPropMap?.[attrName];
        if (!prop) return;

        const def = (Ctor.props || {})[prop];
        let next = coerceFromAttr(def?.type, newValue);
        if (def?.coerce) next = def.coerce(next, this);

        if (this.props[prop]) {
            this.props[prop].value = next;
        }
    }

    #applyStyles() {
        const Ctor = this.constructor;
        const styles = String(Ctor.styles || "");
        if (!styles) return;

        if (
            typeof ShadowRoot !== "undefined" &&
            typeof CSSStyleSheet !== "undefined" &&
            "adoptedStyleSheets" in ShadowRoot.prototype &&
            "replaceSync" in CSSStyleSheet.prototype
        ) {
            if (!Ctor._sharedSheet) {
                Ctor._sharedSheet = new CSSStyleSheet();
                Ctor._sharedSheet.replaceSync(styles);
            }
            this.#root.adoptedStyleSheets = [Ctor._sharedSheet];
            return;
        }

        const styleEl = document.createElement("style");
        styleEl.textContent = styles;
        this.#root.appendChild(styleEl);
    }

    #initProps() {
        const Ctor = this.constructor;
        const defs = Ctor.props || {};

        for (const [prop, def] of Object.entries(defs)) {
            let val = def.default !== undefined ?
                (typeof def.default === "function" ? def.default(this) : def.default) : undefined;
            this.props[prop] = signal(val, { label: `prop_${prop}` });
        }

        for (const [prop, def] of Object.entries(defs)) {
            const attr = def.attr ?? propToAttr(prop);
            if (this.hasAttribute(attr)) {
                const raw = this.getAttribute(attr);
                const coerced = coerceFromAttr(def.type, raw);
                if (coerced !== undefined) {
                    this.props[prop].value = def.coerce ? def.coerce(coerced, this) : coerced;
                }
            }
        }
    }

    #setupReflectionEffects() {
        const defs = this.constructor.props || {};
        for (const [prop, def] of Object.entries(defs)) {
            if (def.reflect) {
                this.runEffect(() => {
                    const val = this.props[prop].value;
                    const attr = def.attr ?? propToAttr(prop);
                    const encoded = coerceToAttr(def.type, val);
                    const current = this.getAttribute(attr);
                    if (current !== encoded) {
                        if (encoded === null) this.removeAttribute(attr);
                        else this.setAttribute(attr, encoded);
                    }
                });
            }
            if (def.watch) {
                let prev = this.props[prop].value;
                this.runEffect(() => {
                    const next = this.props[prop].value;
                    if (next !== prev) {
                        def.watch(next, prev, this);
                        prev = next;
                    }
                });
            }
        }
    }
}

export function h(tag, attrs, ...children) {
    if (!isBrowser) return { tag, attrs, children }; // Stub for SSR
    const el = document.createElement(tag);

    if (attrs) {
        for (const [key, val] of Object.entries(attrs)) {
            const isInlineEventAttr = /^on[a-z]/i.test(key);
            // Event simplified entry
            if (key === "on" && val && typeof val === "object") {
                for (const [evt, listener] of Object.entries(val)) {
                    if (typeof listener !== "function") {
                        if (CONFIG.debug) {
                            console.warn(`[DOM WARNING] Ignored non-function listener for "${evt}".`);
                        }
                        continue;
                    }
                    el.addEventListener(evt, listener);
                }
                continue;
            }
            if (isInlineEventAttr && typeof val === "function") {
                const evt = key[2].toLowerCase() + key.slice(3);
                el.addEventListener(evt, val);
                continue;
            }
            if (key === "on" || isInlineEventAttr) {
                if (CONFIG.debug && val !== null && val !== undefined) {
                    console.warn(`[DOM WARNING] Ignored non-function event binding "${key}". Use a function handler.`);
                }
                continue;
            }
            // Style object support
            if (key === "style" && typeof val === "object") {
                for (const k in val) el.style[k] = val[k];
                continue;
            }
            // Class array support
            if ((key === "class" || key === "className") && Array.isArray(val)) {
                el.setAttribute("class", val.filter(Boolean).join(" "));
                continue;
            }
            if (key === "ref" && typeof val === "function") {
                val(el);
                continue;
            }

            // Sync critical properties
            if (key === "value" || key === "checked" || key === "selected") {
                el[key] = val;
                continue;
            }

            if (val === false || val === null || val === undefined) el.removeAttribute(key);
            else if (val === true) el.setAttribute(key, "");
            else el.setAttribute(key, String(val));
        }
    }

    for (const child of children.flat()) {
        if (child === null || child === undefined || child === false) continue;
        if (child instanceof Node) el.appendChild(child);
        else el.appendChild(document.createTextNode(String(child)));
    }

    return el;
}

export function bindValue(el, sig, options = {}) {
    const {
        event = "input",
        parse = (v) => v,
        format = (v) => (v ?? ""),
        listenerOptions = undefined
    } = options;

            for (const opt of el.options) opt.selected = vals.includes(opt.value);
        } else el.value = val;
    });

    // Event listener for user input
    const listener = (e) => {
        let val;
        if (el.type === "checkbox") val = el.checked;
        else if (el.type === "radio") val = el.value;
        else if (el.multiple && el.tagName === "SELECT") {
            val = Array.from(el.selectedOptions).map(o => o.value);
        } else val = el.value;

        sig.value = parse(val, e);
    };

    el.addEventListener(event, listener, listenerOptions);
    const prevDispose = runner.dispose;
    runner.dispose = () => {
        if (typeof prevDispose === "function") prevDispose();
        el.removeEventListener(event, listener, listenerOptions);
    };

    return runner;
}

/** 
 * Advanced Keyed List Binder.
 * Minimizes DOM moves and preserves state (focus, etc).
 */
export function bindList(parent, itemsSignal, renderItem, options = {}) {
    const keySelector = options.key;
    const wrapFragment = options.wrapFragment ?? "span";
    const cache = new Map(); // Store nodes by key
    const keyOf = (item, index) => {
        if (typeof keySelector === "function") return keySelector(item, index);
        if (typeof keySelector === "string") return item?.[keySelector];
        return index;
    };
    const toNode = (value) => {
        if (!isBrowser) return value;
        if (value instanceof DocumentFragment) {
            if (wrapFragment === false) {
                if (value.childNodes.length === 1) return value.firstChild;
                const msg = "[DOM ERROR] bindList renderItem returned DocumentFragment with multiple roots and wrapFragment=false. Return one stable node or enable wrapping.";
                if (CONFIG.debug) console.warn(msg);
                throw new Error(msg);
            }

            if (CONFIG.debug) {
                console.warn("[DOM WARNING] bindList renderItem returned DocumentFragment. Wrapping into stable container node.");
            }
            const tag = (typeof wrapFragment === "string" && wrapFragment) ? wrapFragment : "span";
            const wrapper = document.createElement(tag);
            if (tag === "span") wrapper.style.display = "contents";
            wrapper.appendChild(value);
            return wrapper;
        }
        if (value instanceof Node) return value;
        if (value === null || value === undefined || value === false) return document.createTextNode("");
        return document.createTextNode(String(value));
    };

    return effect(() => {
        const items = Array.isArray(itemsSignal.value) ? itemsSignal.value : [];
        const newKeys = new Set();
        const nodes = [];

        // 1. Map items to nodes (reuse or create)
        for (let i = 0; i < items.length; i++) {
            const item = items[i];
            const key = keyOf(item, i);
            if (newKeys.has(key)) {
                if (CONFIG.debug) {
                    console.warn(`[DOM WARNING] Duplicate key "${String(key)}" in bindList; keeping first instance.`);
                }
                continue;
            }
            newKeys.add(key);

            let node;
            if (cache.has(key)) {
                node = cache.get(key);
            } else {
                node = toNode(renderItem(item, i));
                cache.set(key, node);
            }
            nodes.push(node);
        }

        // 2. Remove orphaned nodes
        for (const [key, node] of cache.entries()) {
            if (!newKeys.has(key)) {
                if (node.remove) node.remove();
                cache.delete(key);
            }
        }

        // 3. Fast reconcile
        // We use a simplified keyed algorithm that is O(n) for moves.
        let lastNode = null;
        for (let i = 0; i < nodes.length; i++) {
            const node = nodes[i];
            if (node.parentNode !== parent || node.previousSibling !== lastNode) {
                if (lastNode === null) parent.prepend(node);
                else lastNode.after(node);
            }
            lastNode = node;
        }
    });
}

    if (flushCountResetScheduled) return;
    flushCountResetScheduled = true;
    setTimeout(() => {
        flushCountInTick = 0;
        flushCountResetScheduled = false;
    }, 0);
}

function registerFlushRequest() {
    flushCountInTick++;
    scheduleFlushCountReset();
    if (CONFIG.debug && flushCountInTick > CONFIG.warningFlushThreshold) {
        console.warn(`[REACTIVE WARNING] High flush frequency detected (${flushCountInTick} flush requests in one tick). Possible update loop?`);
    }
}

function runEffectQueue() {
    if (processingFlush) return;

    flushScheduled = false;
    processingFlush = true;
    const start = now();

    try {
        while (effectQueue.size) {
            const queue = [...effectQueue];
            effectQueue.clear();

            for (const runner of queue) {
                if (!runner.active) continue;
                if (runner.running) {
                    console.error(`[REACTIVE ERROR] Circular dependency detected in effect: ${runner.label || "anonymous"}`);
                    continue;
                }
                runner();
            }
        }
    } finally {
        processingFlush = false;
        STATS.totalFlushes++;
        STATS.lastFlushMs = now() - start;
        STATS.lastFlushAt = Date.now();
    }
}

/** The Scheduler: deduplicates and batches effect execution */
function flushEffects() {
    if (batchingDepth > 0) return;

    registerFlushRequest();

    if (flushScheduled || processingFlush) return;

    flushScheduled = true;
    queueMicrotask(runEffectQueue);
}

export function isSignal(x) {
    return !!(x && typeof x === "object" && x[SIGNAL_TAG] === true);
}

/** 
 * A Signal is a data box. 
 */
export function signal(initialValue, options = {}) {
    let value = initialValue;
    const subs = new Set();
    const equals = options.equals || Object.is;
    const label = options.label || 'sig';

    STATS.signalsCreated++;

    const out = {
        label,
        [SIGNAL_TAG]: true,
        get value() {
            if (activeEffect) {
                subs.add(activeEffect);
                activeEffect.deps.push(subs);
            }
            return value;
        },
        set value(newVal) {
            if (equals(value, newVal)) return;
            value = newVal;
            if (subs.size === 0) return;
            for (const sub of [...subs]) {
                if (sub.active) effectQueue.add(sub);
            }
            flushEffects();
        },
        peek() { return value; },
        toString() { return String(value); },
        toJSON() { return value; }
    };

    return out;
}

/**
 * An Effect is a reactive job.
 */
export function effect(fn, options = {}) {
    const label = options.label || 'effect';
    let runner;

    // Check if we are outside a component mount/scope and warn if debug is on
    if (CONFIG.debug && !effectTracker && !options.manual) {
        console.warn(`[REACTIVE WARNING] Effect "${label}" created outside ownership. Call this.own() to prevent leaks.`);
    }

    const onCleanup = (cb) => {
        if (typeof cb !== "function") {
            throw new TypeError(`Effect cleanup must be a function [${label}]`);
        }
        runner._cleanup = cb;
    };

    const runRegisteredCleanup = () => {
        if (!runner._cleanup) return;
        try {
            runner._cleanup();
        } catch (e) {
            console.error(`Effect cleanup error [${label}]:`, e);
        } finally {
            runner._cleanup = null;
        }
    };

    runner = () => {
        if (!runner.active) return;

        runRegisteredCleanup();

        cleanup(runner);

        const prev = activeEffect;
        activeEffect = runner;
        runner.running = true; // Loop protection
        try {
            fn(onCleanup);
        } catch (e) {
            console.error(`[REACTIVE ERROR] in effect "${label}":`, e);
            if (runner.stack) console.error("Effect created at:", runner.stack);
            throw e;
        } finally {
            runner.running = false;
            activeEffect = prev;
        }
    };

    runner.active = true;
    runner.running = false;
    runner.label = label;
    runner.deps = [];
    runner._cleanup = null;
    runner.dispose = null;
    if (CONFIG.debug) {
        runner.stack = new Error().stack.split("\n").slice(2).join("\n");
    }

    STATS.effectsCreated++;
    STATS.activeEffects++;

    // Initial run through same runner path
    try {
        runner();
    } catch (e) {
        runner.active = false;
        cleanup(runner);
        STATS.activeEffects = Math.max(0, STATS.activeEffects - 1);
        throw e;
    }

    if (effectTracker) {
        effectTracker(runner);
    }

    return runner;
}

export function cleanup(runner) {
    if (!runner.deps) return;
    for (const subs of runner.deps) subs.delete(runner);
    runner.deps.length = 0;
}

export function stop(runner) {
    if (runner && typeof runner === "object" && typeof runner[COMPUTED_STOP] === "function") {
        runner[COMPUTED_STOP]();
        return;
    }

    if (!runner || !runner.active) return;
    runner.active = false;
    effectQueue.delete(runner);

    if (runner._cleanup) {
        try {
            runner._cleanup();
        } catch (e) {
            console.error(`Effect cleanup error [${runner.label || "effect"}]:`, e);
        } finally {
            runner._cleanup = null;
        }
    }

    if (typeof runner.dispose === "function") {
        try {
            runner.dispose();
        } catch (e) {
            console.error(`Effect dispose error [${runner.label || "effect"}]:`, e);
        } finally {
            runner.dispose = null;
        }
    }

    cleanup(runner);
    STATS.activeEffects = Math.max(0, STATS.activeEffects - 1);
}

export function stopComputed(sig) {
    if (!sig || typeof sig !== "object") return;
    const stopFn = sig[COMPUTED_STOP] || sig.stop;
    if (typeof stopFn === "function") stopFn.call(sig);
}

/**
 * Lazy Computed Signal.
 */
export function computed(fn, options = {}) {
    const equals = options.equals || Object.is;
    const label = options.label || 'computed';
    let value;
    let hasValue = false;
    let dirty = true;
    let computing = false;
    const subs = new Set();

    const stopSelf = () => {
        if (!runner.active) return;
        runner.active = false;
        cleanup(runner);
        for (const sub of [...subs]) {
            if (!sub || !Array.isArray(sub.deps)) continue;
            let idx = sub.deps.indexOf(subs);
            while (idx !== -1) {
                sub.deps.splice(idx, 1);
                idx = sub.deps.indexOf(subs);
            }
        }
        subs.clear();
    };

    const runner = () => {
        if (!runner.active) return;
        if (!dirty) {
            dirty = true;
            if (subs.size === 0) return;
            for (const sub of [...subs]) {
                if (sub.active) effectQueue.add(sub);
            }
            flushEffects();
        }
    };
    runner.active = true;
    runner.running = false;
    runner.label = `${label}_tracker`;
    runner.deps = [];
    runner._cleanup = null;
    runner.dispose = null;
    if (CONFIG.debug) {
        runner.stack = new Error().stack.split("\n").slice(2).join("\n");
    }

    const evaluate = () => {
        if (!runner.active) return value;

        if (computing) {
            const err = new Error(`[REACTIVE ERROR] Circular dependency in computed: ${label}`);
            if (CONFIG.debug && runner.stack) {
                console.error("Computed created at:", runner.stack);
            }
            throw err;
        }

        computing = true;
        cleanup(runner);

        const prev = activeEffect;
        activeEffect = runner;
        runner.running = true;
        try {
            const next = fn();
            if (!hasValue || !equals(value, next)) value = next;
            hasValue = true;
            dirty = false;
        } finally {
            runner.running = false;
            activeEffect = prev;
            computing = false;
        }
    };

    const out = {
        label,
        [SIGNAL_TAG]: true,
        [COMPUTED_STOP]: stopSelf,
        stop: stopSelf,
        peek() {
            if (!runner.active) return value;
            if (dirty) evaluate();
            return value;
        },
        get value() {
            if (!runner.active) return value;
            if (dirty) evaluate();
            if (activeEffect) {
                subs.add(activeEffect);
                activeEffect.deps.push(subs);
            }
            return value;
        }
    };

    return out;
}

    const set = new Set();
    return {
        add(cls) { set.add(cls); },

        values() { return [...set]; }
    };

 * ComponentBuilder lets us chain functions together.
 */
class ComponentBuilder {
    #tag;                // The name of our component (must have a dash like "my-app").
    #props = {};         // The data it can receive.
    #styles = "";        // How it looks.
    #setup = function () { return {}; }; // How it starts up.
    #mount = function () { };           // How it builds its UI.
    #methods = {};       // Things it can DO.
    #onConnected = null; // When it joins the screen.
    #onDisconnected = null; // When it leaves the screen.

    constructor(tag) {
        if (!tag.includes("-")) throw new Error(`Custom element name must include a dash. Got: "${tag}"`);
        this.#tag = tag;
    }

    /** 1. Define what data it takes. */
    props(defs) {
        this.#props = { ...this.#props, ...defs };
        return this;
    }

    /** 2. Define how it looks. */
    styles(styleText) {
        this.#styles = String(styleText ?? "");
        return this;
    }

    /** 3. Define the data and logic (runs once). */
    setup(fn) {
        this.#setup = fn;
        return this;
    }

    /** 4. Define the UI (runs once). */
    mount(fn) {
        this.#mount = fn;
        return this;
    }

    /** 5. Define actions it can perform. */
    methods(methodMap) {
        this.#methods = { ...this.#methods, ...methodMap };
        return this;
    }

    /** 6. Define what happens when it's born. */
    onConnected(fn) {
        this.#onConnected = fn;
        return this;
    }

    /** 7. Define what happens when it dies. */
    onDisconnected(fn) {
        this.#onDisconnected = fn;
        return this;
    }

    /**
     * This final function tells the browser about our new component!
     */
    register() {
        const tag = this.#tag;
        const props = this.#props;
        const styles = this.#styles;
        const setupFn = this.#setup;
        const mountFn = this.#mount;
        const methods = this.#methods;
        const onConnected = this.#onConnected;
        const onDisconnected = this.#onDisconnected;

        // We "bake" all our picked options into a real Class!
        class BuiltComponent extends BaseComponent {
            static props = props;
            static styles = styles;
            static setupFn = setupFn;
            static mountFn = mountFn;
        }

        if (onConnected) BuiltComponent.prototype.onConnected = onConnected;
        if (onDisconnected) BuiltComponent.prototype.onDisconnected = onDisconnected;

        // This makes 'this.someData' act like a shortcut for our Signal's value.
        for (const prop of Object.keys(props)) {
            Object.defineProperty(BuiltComponent.prototype, prop, {
                get() { return this.props[prop].value; },
                set(val) { this.props[prop].value = val; }
            });
        }

        // Put our actions on the component!
        for (const [name, fn] of Object.entries(methods)) {
            if (name in BuiltComponent.prototype) {
                throw new Error(`Method conflicts with existing BaseComponent member: ${name}`);
            }
            BuiltComponent.prototype[name] = fn;
        }

        // Tell the browser!
        if (!customElements.get(tag)) {
            customElements.define(tag, BuiltComponent);
        }

        return BuiltComponent;
    }
}

/** 
 * This is the factory we use to start building!
 */
export const Component = {
    define(tag) { return new ComponentBuilder(tag); }
};

import { signal, effect, stop, setEffectTracker, isSignal, CONFIG } from "./signals.js";
import { bindValue } from "./dom.js";

function coerceFromAttr(type, raw) {
    if (type === "boolean") return raw !== null;
    if (raw === null) return undefined;
    if (type === "number") {
        const n = Number(raw);
        return Number.isFinite(n) ? n : undefined;
    }
    if (type === "json") {
        try { return JSON.parse(raw); } catch { return undefined; }
    }
    return String(raw);
}

function coerceToAttr(type, val) {
    if (type === "boolean") return val ? "" : null;
    if (val === null || val === undefined) return null;
    if (type === "json") return JSON.stringify(val);
    return String(val);
}

function propToAttr(propName) {
    return propName.replace(/[A-Z]/g, (m) => `-${m.toLowerCase()}`);
}

const liveComponents = new Map();

function registerLiveComponent(instance) {
    const tag = instance.localName || instance.constructor?.name || "component";
    let set = liveComponents.get(tag);
    if (!set) {
        set = new Set();
        liveComponents.set(tag, set);
    }
    set.add(instance);
}

function unregisterLiveComponent(instance) {
    const tag = instance.localName || instance.constructor?.name || "component";
    const set = liveComponents.get(tag);
    if (!set) return;
    set.delete(instance);
    if (set.size === 0) liveComponents.delete(tag);
}

export function getComponentStats() {
    const byComponent = {};
    let totalComponents = 0;
    let totalEffects = 0;
    let totalListeners = 0;

    for (const [tag, set] of liveComponents.entries()) {
        let effects = 0;
        let listeners = 0;
        for (const instance of set) {
            const stats = instance.getOwnershipStats();
            effects += stats.effects;
            listeners += stats.listeners;
        }

        byComponent[tag] = { instances: set.size, effects, listeners };
        totalComponents += set.size;
        totalEffects += effects;
        totalListeners += listeners;
    }

    return { totalComponents, totalEffects, totalListeners, byComponent };
}

export class BaseComponent extends HTMLElement {
    #root;
    #mountPoint;
    #effects = new Set();
    #listeners = new Set();
    #mounted = false;
    #setupCtx = null;

    props = Object.create(null);

    static props = {};
    static styles = "";
    static setupFn = null;
    static mountFn = null;

    static get observedAttributes() {
        const defs = this.props || {};
        this._attrToPropMap = {};
        const attrs = [];
        for (const [prop, def] of Object.entries(defs)) {
            const attr = def.attr ?? propToAttr(prop);
            this._attrToPropMap[attr] = prop;
            attrs.push(attr);
        }
        return attrs;
    }

    constructor() {
        super();
        this.#root = this.attachShadow({ mode: "open" });
        this.#mountPoint = document.createElement("div");
        this.#mountPoint.style.display = "contents";
        this.#applyStyles();
        this.#root.appendChild(this.#mountPoint);
        this.#initProps();
    }

    $(selector) { return this.#root.querySelector(selector); }
    $all(selector) { return Array.from(this.#root.querySelectorAll(selector)); }

    emit(name, detail, options = {}) {
        const evt = new CustomEvent(name, {
            detail, bubbles: true, composed: true, cancelable: false, ...options,
        });
        this.dispatchEvent(evt);
        return evt;
    }

    /** Managed Event Listener. Guaranteed cleanup on disconnect. */
    listen(target, type, fn, opts) {
        target.addEventListener(type, fn, opts);
        const unlisten = () => target.removeEventListener(type, fn, opts);
        this.#listeners.add(unlisten);
        return unlisten;
    }

    /** Two-way value binding for components with managed cleanup. */
    bind(el, sig, options) {
        return this.own(bindValue(el, sig, options));
    }

    /**
     * Run code in a tracked scope. Any effects created here belong to this component.
     */
    scope(fn) {
        const prev = setEffectTracker(e => this.#effects.add(e));
        try {
            return fn.call(this, this);
        } finally {
            setEffectTracker(prev);
        }
    }

    provide(context, value) {
        this._providers = this._providers || new Map();
        const sig = isSignal(value) ? value : signal(value);
        this._providers.set(context.name, sig);

        if (!this._contextListenerAttached) {
            this.addEventListener("context-request", (e) => {
                if (this._providers.has(e.detail.name)) {
                    e.stopPropagation();
                    e.detail.callback(this._providers.get(e.detail.name));
                }
            });
            this._contextListenerAttached = true;
        }
    }

    consume(context) {
        let sig = signal(context.initialValue);
        this.dispatchEvent(new CustomEvent("context-request", {
            detail: { name: context.name, callback: (v) => sig = v },
            bubbles: true, composed: true
        }));
        return sig;
    }

    own(runner) {
        this.#effects.add(runner);
        return runner;
    }

    getOwnershipStats() {
        return {
            effects: this.#effects.size,
            listeners: this.#listeners.size
        };
    }

    runEffect(fn) {
        return this.own(effect(fn));
    }

    connectedCallback() {
        if (!this.#mounted) {
            this.#mounted = true;
            this.scope(() => {
                this.#setupReflectionEffects();
                const Ctor = this.constructor;
                this.#setupCtx = Ctor.setupFn ? Ctor.setupFn.call(this, this) : this.props;
                if (Ctor.mountFn) Ctor.mountFn.call(this, this.#mountPoint, this.#setupCtx);
            });
        }
        registerLiveComponent(this);
        if (this.onConnected) this.onConnected();
    }

    disconnectedCallback() {
        const cleanedEffects = this.#effects.size;
        const cleanedListeners = this.#listeners.size;

        // Discipline Check: clear all 
        this.#effects.forEach(e => stop(e));
        this.#effects.clear();

        this.#listeners.forEach(un => un());
        this.#listeners.clear();

        this.#mountPoint.replaceChildren();
        this.#mounted = false;
        unregisterLiveComponent(this);

        if (CONFIG.debug) {
            console.log(`[LIFECYCLE] Component <${this.localName}> destroyed. Cleaned ${cleanedEffects} effects and ${cleanedListeners} listeners.`);
        }

        if (this.onDisconnected) this.onDisconnected();
    }

    attributeChangedCallback(attrName, oldValue, newValue) {
        if (oldValue === newValue) return;
        const Ctor = this.constructor;
        const prop = Ctor._attrToPropMap?.[attrName];
        if (!prop) return;

        const def = (Ctor.props || {})[prop];
        let next = coerceFromAttr(def?.type, newValue);
        if (def?.coerce) next = def.coerce(next, this);

        if (this.props[prop]) {
            this.props[prop].value = next;
        }
    }

    #applyStyles() {
        const Ctor = this.constructor;
        const styles = String(Ctor.styles || "");
        if (!styles) return;

        if (
            typeof ShadowRoot !== "undefined" &&
            typeof CSSStyleSheet !== "undefined" &&
            "adoptedStyleSheets" in ShadowRoot.prototype &&
            "replaceSync" in CSSStyleSheet.prototype
        ) {
            if (!Ctor._sharedSheet) {
                Ctor._sharedSheet = new CSSStyleSheet();
                Ctor._sharedSheet.replaceSync(styles);
            }
            this.#root.adoptedStyleSheets = [Ctor._sharedSheet];
            return;
        }

        const styleEl = document.createElement("style");
        styleEl.textContent = styles;
        this.#root.appendChild(styleEl);
    }

    #initProps() {
        const Ctor = this.constructor;
        const defs = Ctor.props || {};

        for (const [prop, def] of Object.entries(defs)) {
            let val = def.default !== undefined ?
                (typeof def.default === "function" ? def.default(this) : def.default) : undefined;
            this.props[prop] = signal(val, { label: `prop_${prop}` });
        }

        for (const [prop, def] of Object.entries(defs)) {
            const attr = def.attr ?? propToAttr(prop);
            if (this.hasAttribute(attr)) {
                const raw = this.getAttribute(attr);
                const coerced = coerceFromAttr(def.type, raw);
                if (coerced !== undefined) {
                    this.props[prop].value = def.coerce ? def.coerce(coerced, this) : coerced;
                }
            }
        }
    }

    #setupReflectionEffects() {
        const defs = this.constructor.props || {};
        for (const [prop, def] of Object.entries(defs)) {
            if (def.reflect) {
                this.runEffect(() => {
                    const val = this.props[prop].value;
                    const attr = def.attr ?? propToAttr(prop);
                    const encoded = coerceToAttr(def.type, val);
                    const current = this.getAttribute(attr);
                    if (current !== encoded) {
                        if (encoded === null) this.removeAttribute(attr);
                        else this.setAttribute(attr, encoded);
                    }
                });
            }
            if (def.watch) {
                let prev = this.props[prop].value;
                this.runEffect(() => {
                    const next = this.props[prop].value;
                    if (next !== prev) {
                        def.watch(next, prev, this);
                        prev = next;
                    }
                });
            }
        }
    }
}

/** 
 * 'h' helper. Standard DSL for creating elements.
 */
export function h(tag, attrs, ...children) {
    if (!isBrowser) return { tag, attrs, children }; // Stub for SSR
    const el = document.createElement(tag);

    if (attrs) {
        for (const [key, val] of Object.entries(attrs)) {
            const isInlineEventAttr = /^on[a-z]/i.test(key);
            // Event simplified entry
            if (key === "on" && val && typeof val === "object") {
                for (const [evt, listener] of Object.entries(val)) {
                    if (typeof listener !== "function") {
                        if (CONFIG.debug) {
                            console.warn(`[DOM WARNING] Ignored non-function listener for "${evt}".`);
                        }
                        continue;
                    }
                    el.addEventListener(evt, listener);
                }
                continue;
            }
            if (isInlineEventAttr && typeof val === "function") {
                const evt = key[2].toLowerCase() + key.slice(3);
                el.addEventListener(evt, val);
                continue;
            }
            if (key === "on" || isInlineEventAttr) {
                if (CONFIG.debug && val !== null && val !== undefined) {
                    console.warn(`[DOM WARNING] Ignored non-function event binding "${key}". Use a function handler.`);
                }
                continue;
            }
            // Style object support
            if (key === "style" && typeof val === "object") {
                for (const k in val) el.style[k] = val[k];
                continue;
            }
            // Class array support
            if ((key === "class" || key === "className") && Array.isArray(val)) {
                el.setAttribute("class", val.filter(Boolean).join(" "));
                continue;
            }
            if (key === "ref" && typeof val === "function") {
                val(el);
                continue;
            }

            // Sync critical properties
            if (key === "value" || key === "checked" || key === "selected") {
                el[key] = val;
                continue;
            }

            if (val === false || val === null || val === undefined) el.removeAttribute(key);
            else if (val === true) el.setAttribute(key, "");
            else el.setAttribute(key, String(val));
        }
    }

    for (const child of children.flat()) {
        if (child === null || child === undefined || child === false) continue;
        if (child instanceof Node) el.appendChild(child);
        else el.appendChild(document.createTextNode(String(child)));
    }

    return el;
}

/** Comprehensive Input Value Binder */
export function bindValue(el, sig, options = {}) {
    const {
        event = "input",
        parse = (v) => v,
        format = (v) => (v ?? ""),
        listenerOptions = undefined
    } = options;

    // Direct binding for property
    const runner = effect(() => {
        const val = format(sig.value);
        if (el.type === "checkbox") el.checked = !!val;
        else if (el.type === "radio") el.checked = (el.value === String(val));
        else if (el.multiple && el.tagName === "SELECT") {
            const vals = Array.isArray(val) ? val : [val];
            for (const opt of el.options) opt.selected = vals.includes(opt.value);
        } else el.value = val;
    });

    // Event listener for user input
    const listener = (e) => {
        let val;
        if (el.type === "checkbox") val = el.checked;
        else if (el.type === "radio") val = el.value;
        else if (el.multiple && el.tagName === "SELECT") {
            val = Array.from(el.selectedOptions).map(o => o.value);
        } else val = el.value;

        sig.value = parse(val, e);
    };

    el.addEventListener(event, listener, listenerOptions);
    const prevDispose = runner.dispose;
    runner.dispose = () => {
        if (typeof prevDispose === "function") prevDispose();
        el.removeEventListener(event, listener, listenerOptions);
    };

    return runner;
}

export function bindStyle(el, styleSigOrFn) {
    let prev = {};
    return effect(() => {
        const next = readBindingSource(styleSigOrFn);
        const styles = (next && typeof next === "object") ? next : {};

export function bindClass(el, sigMap) {
    let prev = new Set();
    return effect(() => {
        const map = readBindingSource(sigMap);
        const next = new Set();

        if (typeof map === "string") {
            map.split(/\s+/).filter(Boolean).forEach(cls => next.add(cls));
        } else if (Array.isArray(map)) {
            map.filter(Boolean).forEach(cls => next.add(String(cls)));
        } else if (map && typeof map === "object") {
            for (const [cls, active] of Object.entries(map)) {
                if (active) next.add(cls);
            }
        }

        for (const cls of prev) {
            if (!next.has(cls)) el.classList.remove(cls);
        }
        for (const cls of next) {
            if (!prev.has(cls)) el.classList.add(cls);
        }

        prev = next;
    });
}

/** 
 * Advanced Keyed List Binder.
 * Minimizes DOM moves and preserves state (focus, etc).
 */
export function bindList(parent, itemsSignal, renderItem, options = {}) {
    const keySelector = options.key;
    const wrapFragment = options.wrapFragment ?? "span";
    const cache = new Map(); // Store nodes by key
    const keyOf = (item, index) => {
        if (typeof keySelector === "function") return keySelector(item, index);
        if (typeof keySelector === "string") return item?.[keySelector];
        return index;
    };
    const toNode = (value) => {
        if (!isBrowser) return value;
        if (value instanceof DocumentFragment) {
            if (wrapFragment === false) {
                if (value.childNodes.length === 1) return value.firstChild;
                const msg = "[DOM ERROR] bindList renderItem returned DocumentFragment with multiple roots and wrapFragment=false. Return one stable node or enable wrapping.";
                if (CONFIG.debug) console.warn(msg);
                throw new Error(msg);
            }

            if (CONFIG.debug) {
                console.warn("[DOM WARNING] bindList renderItem returned DocumentFragment. Wrapping into stable container node.");
            }
            const tag = (typeof wrapFragment === "string" && wrapFragment) ? wrapFragment : "span";
            const wrapper = document.createElement(tag);
            if (tag === "span") wrapper.style.display = "contents";
            wrapper.appendChild(value);
            return wrapper;
        }
        if (value instanceof Node) return value;
        if (value === null || value === undefined || value === false) return document.createTextNode("");
        return document.createTextNode(String(value));
    };

    return effect(() => {
        const items = Array.isArray(itemsSignal.value) ? itemsSignal.value : [];
        const newKeys = new Set();
        const nodes = [];

        // 1. Map items to nodes (reuse or create)
        for (let i = 0; i < items.length; i++) {
            const item = items[i];
            const key = keyOf(item, i);
            if (newKeys.has(key)) {
                if (CONFIG.debug) {
                    console.warn(`[DOM WARNING] Duplicate key "${String(key)}" in bindList; keeping first instance.`);
                }
                continue;
            }
            newKeys.add(key);

            let node;
            if (cache.has(key)) {
                node = cache.get(key);
            } else {
                node = toNode(renderItem(item, i));
                cache.set(key, node);
            }
            nodes.push(node);
        }

        // 2. Remove orphaned nodes
        for (const [key, node] of cache.entries()) {
            if (!newKeys.has(key)) {
                if (node.remove) node.remove();
                cache.delete(key);
            }
        }

        // 3. Fast reconcile
        // We use a simplified keyed algorithm that is O(n) for moves.
        let lastNode = null;
        for (let i = 0; i < nodes.length; i++) {
            const node = nodes[i];
            if (node.parentNode !== parent || node.previousSibling !== lastNode) {
                if (lastNode === null) parent.prepend(node);
                else lastNode.after(node);
            }
            lastNode = node;
        }
    });

 * and built-in diagnostics.

// Global configuration
export const CONFIG = {
    debug: false,
    warningFlushThreshold: 100 // Warnings if flushing too often in one tick
};

let activeEffect = null;
let effectQueue = new Set();
let flushScheduled = false;
let processingFlush = false;
let batchingDepth = 0;
let flushCountInTick = 0;
let flushCountResetScheduled = false;

const SIGNAL_TAG = Symbol.for("stankovich.signal");
const COMPUTED_STOP = Symbol.for("stankovich.computed.stop");

const STATS = {
    signalsCreated: 0,
    effectsCreated: 0,
    activeEffects: 0,
    totalFlushes: 0,
    lastFlushMs: 0,
    lastFlushAt: 0
};

const now = () => (
    typeof performance !== "undefined" && typeof performance.now === "function"
        ? performance.now()
        : Date.now()
);

export function getReactiveStats() {
    return {
        ...STATS,
        queuedEffects: effectQueue.size
    };
}

// Tracker stack for nested component mounting
let effectTracker = null;
export function setEffectTracker(fn) {
    const prev = effectTracker;
    effectTracker = fn;
    return prev;
}

/** Group multiple signal changes into one single update. */
export function batch(fn) {
    batchingDepth++;
    try {
        fn();
    } finally {
        batchingDepth--;
        if (batchingDepth === 0) flushEffects();
    }
}

/** Run code without creating signal dependencies. */
export function untracked(fn) {
    const prev = activeEffect;
    activeEffect = null;
    try {
        return fn();
    } finally {
        activeEffect = prev;
    }
}

function scheduleFlushCountReset() {
    if (flushCountResetScheduled) return;
    flushCountResetScheduled = true;
    setTimeout(() => {
        flushCountInTick = 0;
        flushCountResetScheduled = false;
    }, 0);
}

function registerFlushRequest() {
    flushCountInTick++;
    scheduleFlushCountReset();
    if (CONFIG.debug && flushCountInTick > CONFIG.warningFlushThreshold) {
        console.warn(`[REACTIVE WARNING] High flush frequency detected (${flushCountInTick} flush requests in one tick). Possible update loop?`);
    }
}

function runEffectQueue() {
    if (processingFlush) return;

    flushScheduled = false;
    processingFlush = true;
    const start = now();

    try {
        while (effectQueue.size) {
            const queue = [...effectQueue];
            effectQueue.clear();

            for (const runner of queue) {
                if (!runner.active) continue;
                if (runner.running) {
                    console.error(`[REACTIVE ERROR] Circular dependency detected in effect: ${runner.label || "anonymous"}`);
                    continue;
                }
                runner();
            }
        }
    } finally {
        processingFlush = false;
        STATS.totalFlushes++;
        STATS.lastFlushMs = now() - start;
        STATS.lastFlushAt = Date.now();
    }
}

/** The Scheduler: deduplicates and batches effect execution */
function flushEffects() {
    if (batchingDepth > 0) return;

    registerFlushRequest();

    if (flushScheduled || processingFlush) return;

    flushScheduled = true;
    queueMicrotask(runEffectQueue);
}

export function isSignal(x) {
    return !!(x && typeof x === "object" && x[SIGNAL_TAG] === true);
}

/** 
 * A Signal is a data box. 
 */
export function signal(initialValue, options = {}) {
    let value = initialValue;
    const subs = new Set();
    const equals = options.equals || Object.is;
    const label = options.label || 'sig';

    STATS.signalsCreated++;

    const out = {
        label,
        [SIGNAL_TAG]: true,
        get value() {
            if (activeEffect) {
                subs.add(activeEffect);
                activeEffect.deps.push(subs);
            }
            return value;
        },
        set value(newVal) {
            if (equals(value, newVal)) return;
            value = newVal;
            if (subs.size === 0) return;
            for (const sub of [...subs]) {
                if (sub.active) effectQueue.add(sub);
            }
            flushEffects();
        },
        peek() { return value; },
        toString() { return String(value); },
        toJSON() { return value; }
    };

    return out;
}

/**
 * An Effect is a reactive job.
 */
export function effect(fn, options = {}) {
    const label = options.label || 'effect';
    let runner;

    // Check if we are outside a component mount/scope and warn if debug is on
    if (CONFIG.debug && !effectTracker && !options.manual) {
        console.warn(`[REACTIVE WARNING] Effect "${label}" created outside ownership. Call this.own() to prevent leaks.`);
    }

    const onCleanup = (cb) => {
        if (typeof cb !== "function") {
            throw new TypeError(`Effect cleanup must be a function [${label}]`);
        }
        runner._cleanup = cb;
    };

    const runRegisteredCleanup = () => {
        if (!runner._cleanup) return;
        try {
            runner._cleanup();
        } catch (e) {
            console.error(`Effect cleanup error [${label}]:`, e);
        } finally {
            runner._cleanup = null;
        }
    };

    runner = () => {
        if (!runner.active) return;

        runRegisteredCleanup();

        cleanup(runner);

        const prev = activeEffect;
        activeEffect = runner;
        runner.running = true; // Loop protection
        try {
            fn(onCleanup);
        } catch (e) {
            console.error(`[REACTIVE ERROR] in effect "${label}":`, e);
            if (runner.stack) console.error("Effect created at:", runner.stack);
            throw e;
        } finally {
            runner.running = false;
            activeEffect = prev;
        }
    };

    runner.active = true;
    runner.running = false;
    runner.label = label;
    runner.deps = [];
    runner._cleanup = null;
    runner.dispose = null;
    if (CONFIG.debug) {
        runner.stack = new Error().stack.split("\n").slice(2).join("\n");
    }

    STATS.effectsCreated++;
    STATS.activeEffects++;

    // Initial run through same runner path
    try {
        runner();
    } catch (e) {
        runner.active = false;
        cleanup(runner);
        STATS.activeEffects = Math.max(0, STATS.activeEffects - 1);
        throw e;
    }

    if (effectTracker) {
        effectTracker(runner);
    }

    return runner;
}

export function cleanup(runner) {
    if (!runner.deps) return;
    for (const subs of runner.deps) subs.delete(runner);
    runner.deps.length = 0;
}

export function stop(runner) {
    if (runner && typeof runner === "object" && typeof runner[COMPUTED_STOP] === "function") {
        runner[COMPUTED_STOP]();
        return;
    }

    if (!runner || !runner.active) return;
    runner.active = false;
    effectQueue.delete(runner);

    if (runner._cleanup) {
        try {
            runner._cleanup();
        } catch (e) {
            console.error(`Effect cleanup error [${runner.label || "effect"}]:`, e);
        } finally {
            runner._cleanup = null;
        }
    }

    if (typeof runner.dispose === "function") {
        try {
            runner.dispose();
        } catch (e) {
            console.error(`Effect dispose error [${runner.label || "effect"}]:`, e);
        } finally {
            runner.dispose = null;
        }
    }

    cleanup(runner);
    STATS.activeEffects = Math.max(0, STATS.activeEffects - 1);
}

export function stopComputed(sig) {
    if (!sig || typeof sig !== "object") return;
    const stopFn = sig[COMPUTED_STOP] || sig.stop;
    if (typeof stopFn === "function") stopFn.call(sig);
}

/**
 * Lazy Computed Signal.
 */
export function computed(fn, options = {}) {
    const equals = options.equals || Object.is;
    const label = options.label || 'computed';
    let value;
    let hasValue = false;
    let dirty = true;
    let computing = false;
    const subs = new Set();

    const stopSelf = () => {
        if (!runner.active) return;
        runner.active = false;
        cleanup(runner);
        for (const sub of [...subs]) {
            if (!sub || !Array.isArray(sub.deps)) continue;
            let idx = sub.deps.indexOf(subs);
            while (idx !== -1) {
                sub.deps.splice(idx, 1);
                idx = sub.deps.indexOf(subs);
            }
        }
        subs.clear();
    };

    const runner = () => {
        if (!runner.active) return;
        if (!dirty) {
            dirty = true;
            if (subs.size === 0) return;
            for (const sub of [...subs]) {
                if (sub.active) effectQueue.add(sub);
            }
            flushEffects();
        }
    };
    runner.active = true;
    runner.running = false;
    runner.label = `${label}_tracker`;
    runner.deps = [];
    runner._cleanup = null;
    runner.dispose = null;
    if (CONFIG.debug) {
        runner.stack = new Error().stack.split("\n").slice(2).join("\n");
    }

    const evaluate = () => {
        if (!runner.active) return value;

        if (computing) {
            const err = new Error(`[REACTIVE ERROR] Circular dependency in computed: ${label}`);
            if (CONFIG.debug && runner.stack) {
                console.error("Computed created at:", runner.stack);
            }
            throw err;
        }

        computing = true;
        cleanup(runner);

        const prev = activeEffect;
        activeEffect = runner;
        runner.running = true;
        try {
            const next = fn();
            if (!hasValue || !equals(value, next)) value = next;
            hasValue = true;
            dirty = false;
        } finally {
            runner.running = false;
            activeEffect = prev;
            computing = false;
        }
    };

    const out = {
        label,
        [SIGNAL_TAG]: true,
        [COMPUTED_STOP]: stopSelf,
        stop: stopSelf,
        peek() {
            if (!runner.active) return value;
            if (dirty) evaluate();
            return value;
        },
        get value() {
            if (!runner.active) return value;
            if (dirty) evaluate();
            if (activeEffect) {
                subs.add(activeEffect);
                activeEffect.deps.push(subs);
            }
            return value;
        }
    };

    return out;
}

#!/usr/bin/env bash
set -euo pipefail

usage() {
  cat >&2 <<'EOF'
usage: merge-files.sh <target-dir> [--exclude=png,jpg] [--include=md,yaml] [--dry-run]
EOF
}

declare -A ignored_dirs=(
  ["vendor"]=1
  ["node_modules"]=1
  ["storage"]=1
  ["tmp"]=1
  ["dist"]=1
  ["build"]=1
  ["coverage"]=1
  ["__pycache__"]=1
  [".idea"]=1
  [".git"]=1
  [".polymoly"]=1
  [".cache"]=1
  ["artifacts"]=1
)

declare -A exclude_exts=()
declare -A include_exts=()

target_dir=""
dry_run=0

add_csv_to_set() {
  local csv="$1"
  local -n set_ref="$2"
  local value

  IFS=',' read -r -a values <<< "$csv"
  for value in "${values[@]}"; do
    value="${value#"${value%%[![:space:]]*}"}"
    value="${value%"${value##*[![:space:]]}"}"
    value="${value#.}"
    if [[ -n "$value" ]]; then
      set_ref["$value"]=1
    fi
  done
}

while (($# > 0)); do
  case "$1" in
    --exclude=*)
      add_csv_to_set "${1#*=}" exclude_exts
      shift
      ;;
    --exclude)
      [[ $# -ge 2 ]] || { usage; exit 2; }
      add_csv_to_set "$2" exclude_exts
      shift 2
      ;;
    --include=*)
      add_csv_to_set "${1#*=}" include_exts
      shift
      ;;
    --include)
      [[ $# -ge 2 ]] || { usage; exit 2; }
      add_csv_to_set "$2" include_exts
      shift 2
      ;;
    --dry-run)
      dry_run=1
      shift
      ;;
    --help|-h)
      usage
      exit 0
      ;;
    --*)
      printf 'unknown option: %s\n' "$1" >&2
      usage
      exit 2
      ;;
    *)
      if [[ -n "$target_dir" ]]; then
        usage
        exit 2
      fi
      target_dir="$1"
      shift
      ;;
  esac
done

[[ -n "$target_dir" ]] || { usage; exit 2; }
[[ -d "$target_dir" ]] || { printf "directory '%s' does not exist\n" "$target_dir" >&2; exit 1; }

if ((${#exclude_exts[@]} > 0 && ${#include_exts[@]} > 0)); then
  printf 'you cannot use --exclude and --include at the same time\n' >&2
  exit 1
fi

root="$(realpath "$target_dir")"
output_file="$root/$(basename "$root").txt"

printf 'Scanning directory: %s\n' "$root"
printf 'Output file: %s\n' "$output_file"
printf 'Ignoring directories:\n'
printf '%s\n' "${!ignored_dirs[@]}" | LC_ALL=C sort | sed 's/^/- /'
printf '%s\n' '----------------------------------------'

get_extension_key() {
  local path="$1"
  local base ext
  base="$(basename "$path")"

  if [[ "$base" != *.* ]]; then
    printf '%s' "$base"
    return
  fi

  ext="${base##*.}"
  if [[ "$ext" == "$base" ]]; then
    printf '%s' "$base"
  else
    printf '%s' "$ext"
  fi
}

is_text_file() {
  local path="$1"

  if [[ ! -s "$path" ]]; then
    return 0
  fi

  LC_ALL=C grep -Iq . "$path"
}

mapfile -t files < <(
  find "$root" \
    \( -path "$root/vendor" -o -path "$root/vendor/*" \
       -o -path "$root/node_modules" -o -path "$root/node_modules/*" \
       -o -path "$root/storage" -o -path "$root/storage/*" \
       -o -path "$root/tmp" -o -path "$root/tmp/*" \
       -o -path "$root/dist" -o -path "$root/dist/*" \
       -o -path "$root/build" -o -path "$root/build/*" \
       -o -path "$root/coverage" -o -path "$root/coverage/*" \
       -o -path "$root/__pycache__" -o -path "$root/__pycache__/*" \
       -o -path "$root/.idea" -o -path "$root/.idea/*" \
       -o -path "$root/.git" -o -path "$root/.git/*" \
       -o -path "$root/.polymoly" -o -path "$root/.polymoly/*" \
       -o -path "$root/.cache" -o -path "$root/.cache/*" \
       -o -path "$root/artifacts" -o -path "$root/artifacts/*" \) -prune \
    -o -type f -print | LC_ALL=C sort
)

merged=0
skipped=0
tmp_output=""

if ((dry_run == 0)); then
  tmp_output="$(mktemp)"
fi

cleanup() {
  if [[ -n "$tmp_output" && -f "$tmp_output" ]]; then
    rm -f "$tmp_output"
  fi
}
trap cleanup EXIT

for path in "${files[@]}"; do
  rel="${path#"$root"/}"

  if [[ "$path" == "$output_file" ]]; then
    printf 'Skipping (generated output file): %s\n' "$rel"
    ((skipped+=1))
    continue
  fi

  ext="$(get_extension_key "$path")"

  if ((${#include_exts[@]} > 0)) && [[ -z "${include_exts[$ext]+x}" ]]; then
    printf 'Skipping (not in include list): %s\n' "$rel"
    ((skipped+=1))
    continue
  fi

  if ((${#include_exts[@]} == 0)) && [[ -n "${exclude_exts[$ext]+x}" ]]; then
    printf 'Skipping (excluded by default or option): %s\n' "$rel"
    ((skipped+=1))
    continue
  fi

  if ! is_text_file "$path"; then
    printf 'Skipping (binary or non-text): %s\n' "$rel"
    ((skipped+=1))
    continue
  fi

  ((merged+=1))
  if ((dry_run == 1)); then
    printf '[DRY-RUN] Would merge: %s\n' "$rel"
    continue
  fi

  printf 'Merging: %s\n' "$rel"
  {
    printf '=== %s ===\n' "$rel"
    cat "$path"
    if [[ -s "$path" ]]; then
      last_byte="$(tail -c 1 "$path" | od -An -tx1 | tr -d ' \n')"
      if [[ "$last_byte" != "0a" ]]; then
        printf '\n'
      fi
    fi
    printf '\n'
  } >> "$tmp_output"
done

if ((dry_run == 0)); then
  mv "$tmp_output" "$output_file"
  tmp_output=""
fi

printf '%s\n' '----------------------------------------'
printf 'Done!\n'
printf 'Merged files : %d\n' "$merged"
printf 'Skipped files: %d\n' "$skipped"
printf 'Output file  : %s\n' "$output_file"

    const set = new Set();
    return {
        add(cls) { set.add(cls); },
        remove(cls) { set.delete(cls); },
        has(cls) { return set.has(cls); },
        values() { return [...set]; }
    };
}
